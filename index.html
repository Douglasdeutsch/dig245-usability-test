The Setup


The site presents the results of a usability test on the im leagues website. We conducted the test using five tasks, each of which was chosen because a typical user would be likely to try these tasks.

Task 1

Login

This took 1 minute. User had some login trouble, but website was slow and filled with ads.

Task 2

Find and list the active IM teams for Davidson Fall 2022

User took 5 extra clicks. 40 seconds of time.

Task 3

Find the flickerball team from your freshman hall playing this fall

It took 1 minute for the user to navigate to the correct page

Task 4

Find the receiving touchdown leader for IM flag football Fall 2022

1 minute (user took different path than we anticipated)

Conclusion

Based on the results of the usability test, the IM leagues website is lacking in several areas of website usability. Navigation is particularly hard, and the accessibility features are minimal. There are multiple paths to get to a desired location, which are not necessary. The amount of popup ads make the website an unenjoyable experience.
